Parameters:

  • fs_methods ‚Äì List of feature selection methods (classes or strings).

  • merging_strategy ‚Äì Merging strategy (class or string).

  • num_repeats ‚Äì Number of repeats for the pipeline.

  • num_features_to_select ‚Äì Desired number of features to select. (For rank-based merging, this must be provided.)

  • metrics ‚Äì List of performance metrics (classes or strings).

  • task ‚Äì ‚Äòclassification‚Äô or ‚Äòregression‚Äô.

  • min_group_size ‚Äì Minimum number of methods in each subgroup.

  • fill ‚Äì If True, ensures exactly num_features_to_select features.

  • random_state ‚Äì Seed for reproducibility.

  • n_jobs ‚Äì Number of parallel jobs (-1 uses all processors).

Raises:

ValueError ‚Äì If the task is invalid or if num_features_to_select is missing for rank-based merging.
