__init__(data, fs_methods, merging_strategy, num_repeats, num_features_to_select, metrics=['logloss', 'f1_score', 'accuracy'], task='classification', min_group_size=2, random_state=None, n_jobs=None)[source]ÔÉÅ

Initializes a FeatureSelectionPipeline object, which configures and executes a feature selection process using specified methods and strategies.

Parameters:

  • data (pandas.DataFrame) ‚Äì The dataset to process, which must include a ‚Äòtarget‚Äô column.

  • fs_methods (list of classes or str) ‚Äì Feature selection methods to be used. Each entry can be either a class or a string that corresponds to a defined class name. These classes should implement a method for feature selection.

  • merging_strategy (class or str) ‚Äì The strategy used to merge features selected by different methods. Can be either a class or a string corresponding to a defined class name.

  • num_repeats (int) ‚Äì The number of times the feature selection process will be repeated.

  • threshold (int, optional) ‚Äì The number of features to select. Defaults to one-tenth of the number of features in ‚Äòdata‚Äô if not provided.

  • metrics (list of classes or str) ‚Äì Performance metrics to be used for evaluation. Each entry can be either a class or a string that corresponds to a defined class name. These classes should implement a method for computing the metric.

  • task (str, optional) ‚Äì The type of task (‚Äòclassification‚Äô or ‚Äòregression‚Äô). Defaults to ‚Äòclassification‚Äô.

  • random_state (int, optional) ‚Äì Random seed for reproducibility. Defaults to None.

  • n_jobs (int, optional) ‚Äì The number of jobs to compute the feature selection pipeline, -1 means using all processors.

Raises:

ValueError ‚Äì If ‚Äòtask‚Äô is neither ‚Äòclassification‚Äô nor ‚Äòregression‚Äô.

Example

>>> pipeline = FeatureSelectionPipeline(
    data=df,
    fs_methods=['RandomForestSelector', 'MutualInfoSelector'],
    merging_strategy='BordaMerger',
    num_repeats=10,
    threshold=5,
    metrics=['loglloss', 'f1_score', 'accuracy'],
    task='classification',
    random_state=42
    n_jobs=n_jobs
)

run(verbose=True)[source]ÔÉÅ

Runs the entire feature selection and classification pipeline.

This method iterates through the pipeline for the specified number of repeats, performing: - Data splitting - Feature computation and merging - Performance evaluation - Calculation of mean metrics per repeat - Pareto analysis to identify best performing groups and repeats

Returns: - tuple: A tuple containing the merged features, the best repeat, and the best group name.
